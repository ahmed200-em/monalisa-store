## ⚙️ **Quick Customization**

### **Change Products:**

### **Change Delivery Costs:**
Edit `livraison.txt` → Update city pricing

### **Change Admin Password:**
Edit `admin.js` → Line 6: `this.adminPassword = 'your-new-password'`

### **Change Store Info:**
Edit `store-details.txt` → Update contact information
